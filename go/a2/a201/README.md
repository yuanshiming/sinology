

##### 國語
`校對：luo`

* * *

春秋時左丘明撰，二十一卷。分別記載周、魯、齊、晉、鄭、楚、吳、越等八國的事蹟，自周穆王起，至魯悼公止，共歷五百餘年，為一分國紀事之史。或稱為「春秋外傳」。
〈資料來源：國語辭典〉
