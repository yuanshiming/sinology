

##### 戰國策
　　　`鮑本為主``校對：yny`

* * *

漢劉向輯，三十三卷。依國別分為西周、東周、秦、齊、楚、趙、魏、韓、燕、宋、衛、中山十二策。內容記載戰國時代各諸侯國的政治大事及當時策士的言論行動。描寫人事，運用誇飾、比喻、排比等手法，語言形象鮮明，辯辭宏肆瑰麗，刻劃歷史人物亦活躍生動，為我國重要的史料。今通行有漢高誘注本。或稱為「短長書」、「國策」、「長書」。
〈資料來源：國語辭典〉
